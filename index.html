Jonas Nilsson Front End Developer

Som front end utvecklare kan jag erbjuda en mängd olika tjänster beroende på ditt behov. Jag kan bygga webbplatser med responsiv design som fungerar sömlöst på olika enheter och plattformar, skapa användarvänliga gränssnitt som gör det enkelt för dina användare att navigera på din webbplats. Jag kan även hjälpa dig att uppdatera och förbättra befintliga webbplatser med att integrera nya funktioner och tekniker.

Photos - API

Inlämningsuppgift i API kursen. Backend för ett foto REST-API. Skrivet i Typescript, använder Prisma som ORM till MySQL databasen. Node ihop med experss sköter routingen med totalt 16 endpoints och inkommande requests valideras med express-validator. Använder sig av JWT, JSON Web Token, för autentisering med både access- och refresh-tokens för att kontrollera om det är rätt användare som gör request.
